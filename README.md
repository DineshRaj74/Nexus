# Nexus

# Nexus Digitalia – Corporate Website

A fully responsive corporate website developed for *Nexus Digitalia*, a software development company, during my *Web Development Internship*.

The website showcases the company's services, expertise, software development process, testimonials, FAQs, and contact information through a modern and responsive user interface.

> *Note:* This project was developed as a real-time internship project for learning and implementing front-end web development concepts.

---
# Features

- Modern responsive UI
- Bootstrap 5 Responsive Navigation
- Offcanvas Mobile Menu
- Hero Section
- Company Overview Section
- Interactive Service Tabs
- Software Development Process
- Company Highlights
- Testimonial Carousel
- FAQ Accordion
- Contact Form
- WhatsApp Integration
- Email Integration
- Floating Social Media Sidebar
- Back to Top Button
- Responsive Footer
- Mobile, Tablet and Desktop Optimized
